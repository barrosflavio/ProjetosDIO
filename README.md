# Projetos e Desafios DIO
Este repositório apenas irá INDEXAR os links do meu Gitea ("Github" Selfhosted) pois com o passar do tempo há muitos projetos basicos e pouco atrativos poluindo o meu Github devido a quantidade de cursos diferentes que vou fazendo. Pensando nisso eu estou auto-hospedando um "Github" em meu servidor, usando um serviço Open Source chamado Gitea, porém como a DIO se recusa a aceitar os links de repositório do meu Gitea que são tipo: https://gitea.phpense.com/DIO/NomeDoRepositorio. Estarei apontando todas as minhas entregas, de desafios ou projetos da DIO para este repositorio Github, indexando os links dos meus repositórios auto-hospedados neste Readme e comunicando na descrição de entrega da platafoma DIO quando vamos finalizar o requisito/desafio.

Caso ninguém leia a descrição de entrega, e apenas abra o link que trará para este repositório, o mesmo descobrirá que basta procurar o desafio ou projeto em questão logo abaixo, ou ver o link correto na descrição de entrega.

Agradeço a todos pela compreensão. #GithubSemTralha
